Install webpack support
=======================
npm install webpack webpack-cli --save-dev

Build TypeScript code with Visual Studio Code
=============================================
Windows: ctrl+shit+b
Mac: command+shit+b
Or open terminal and and run: tsc (by default tsc will use tsconfig.json)

Build webpack
=============
npx webpack

Run web server
==============
npm install express (just do this one time)
node ./server.js

Run HelloWorld in web browser
=============================
http://localhost:3000/dist/index.html
